And now we can run a query and immediately plot a visualization of the result using Pandas plotting syntax!

c.sql("""
    SELECT floor(fdi) AS fdi, avg(gdp) as gdp
    FROM fiscal_cached 
    WHERE fdi > 50 AND gdp >= 0 
    GROUP BY floor(fdi)
""").compute().plot(x='fdi', y='gdp')
